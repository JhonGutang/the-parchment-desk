# 🪶 The Parchment Desk

_The magical, whimsical resume builder – just for fun!_

Live: [https://frontend-z0en.onrender.com/](https://frontend-z0en.onrender.com/)

## What is The Parchment Desk?

The Parchment Desk lets you conjure up enchanting resumes inspired by old-world wizardry and mystical academia. Enter your credentials, pass the magical verification, and craft a parchment ready to impress wizards, witches, and muggles alike.

## Features

- ✨ Email-based enchanted portal login
- 📜 Styled "parchment" resume export
- 🦉 Whimsical, theme-rich UI
- 🪄 Powered by modern Typescript & tRPC stack

## Local Development

See `local/AGENTS.md` and project documentation for running, logs, and config.

1. Install deps: `pnpm install`
2. Start dev: `zap start`
3. Check logs: `zap logs`
4. Visit [localhost] as prompted
